\# 🔌 Configuração dos MCPs no Cline



\## 📌 Passo a Passo



1\. Abra o VS Code.

2\. Clique no ícone do \*\*Cline\*\* na barra lateral.

3\. Clique no ícone de \*\*Engrenagem\*\* → \*\*MCP Servers\*\*.

4\. Clique no botão para abrir `cline\_mcp\_settings.json`.

5\. Cole o conteúdo do arquivo `cline\_mcp\_settings.json` fornecido.



\## 🔧 Verificação



Após configurar, verifique se todos os servidores estão com status \*\*"Conectado"\*\* (indicador verde).



\## 📌 Servidores MCP Ativos



| Servidor | Função |

| :--- | :--- |

| `sequential-thinking` | Pensamento lógico em etapas |

| `codecompress` | Compacta código para economizar tokens |

| `pg\_curso` | Conecta ao PostgreSQL |

| `playwright` | Testa interface no navegador |

| `filesystem` | Acesso seguro a arquivos |

| `glider` | Análise estática de código C# |

| `dotnet-build` | Compila o projeto via MCP |

| `windows-search` | Busca e substitui em lote |
